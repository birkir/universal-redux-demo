# UENO. Starter kit

Starter kit with server side rendering, eslint, stylus, stylelint and enzyme tests.

**Before working on a new project**

Change stuff in `app.json` and `package.json`.

## Install

```bash
$ npm install
```

## Run dev
```bash
$ npm run dev
```

## Run release
```bash
$ npm start
```
